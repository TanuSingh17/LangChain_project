Restaurant Name & Menu Generator
A simple beginner-friendly AI project built using Streamlit, LangChain, and OpenAI.

This application generates:
- Creative restaurant names
- Suggested menu items
based on the cuisine selected by the user using Large Language Models (LLMs).

Features:
- Simple and interactive Streamlit UI
- AI-generated restaurant names
- AI-generated menu items
- Uses LangChain prompt templates and chains
- Beginner-friendly GenAI project

Tech Stack:
- Python
- Streamlit
- LangChain
- OpenAI API

Installation
1. Clone the Repository
git clone https://github.com/TanuSingh17/LangChain_project.git
cd LangChain_project

2. Install Dependencies:
pip install streamlit
pip install langchain
pip install langchain-openai
pip install openai

3. Add OpenAI API Key
Create a file named:
secret_key.py
Add:
openapi_key = "YOUR_OPENAI_API_KEY"

Run the Application
streamlit run main.py
